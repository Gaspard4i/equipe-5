# Expliquez les difficultés techniques auxquelles vous avez été confrontés et comment vous les avez surmontées

- Séparation client/server : nous avions déjà beaucoup avancé, mais il fait tout séparé en client/server, nous avons donc pris le code petit bout par petit pour le séparer en client/server.
- Tests et boucle TDD : nous avons eu des deifficultés à trouver quoi tester, nous avons donc beaucoup de test pour la class Player.
- Trouver les calculs, les optimiser. Rechercher sur internet ou grâce à des calculs d'ancien projet on a pu trouver les bons calculs.
- Au début gérer le partages des tâches , aux final tout le monde a bien travaillé et ça s'est arrangé tout seul
- Avoir des bots 'intelligents', on avait des bots qui bougeait aléatoirement sur place au final avec un peu de persévérance et de teste on a réussit à les rendre 'intelligents'

# Indiquez les points d'amélioration/d'achèvement de votre projet

- Mieux séparer les issues : l'issue server rà duré 1 jour on aurait pu la séparer en plusieurs issues.
- Mieux gérer les tests : faire plus de tests et plus de TDD
- Organisation server/client : mieux séparer les deux parties
- Organisation du code : mieux separer le codes en fichiers, bien rangés en dossiers
- Organisation interface : être plus érgonomique
- Respecter le cahier des charges : nous n'avons pas réaliser tout le cahier des charges

# Enfin, expliquez ce dont vous êtes le plus fier/fière

Nous sommes premièrement fier d'avoir un jeu jouable et très semblable à un vrai jeu.
Nous sommes très fier de notre visuel, notamment les animations qui ajoutenet un plus.
Nous avons eu aucun problème au niveau de la gestion d'équipe, aucune embrouille, tout le monde a bien travaillé.
